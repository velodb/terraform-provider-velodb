# velodb_clusters (Data Source)

Use the *velodb_clusters* data source to list and filter clusters in a VeloDB Cloud warehouse.

## Example Usage

### List All Clusters in a Warehouse

```terraform
data "velodb_clusters" "all" {
  warehouse_id = "ALBJRXRG"
}

output "cluster_names" {
  value = [for cl in data.velodb_clusters.all.clusters : cl.name]
}
```

### Filter Running Compute Clusters

```terraform
data "velodb_clusters" "running_compute" {
  warehouse_id = velodb_warehouse.main.id
  status       = "Running"
  cluster_type = "COMPUTE"
}
```

### Filter by Pay Type

```terraform
data "velodb_clusters" "postpaid" {
  warehouse_id = velodb_warehouse.main.id
  pay_type     = "PostPaid"
}
```
