# velodb_warehouses (Data Source)

Use the *velodb_warehouses* data source to list and filter warehouses in your VeloDB Cloud organization.

## Example Usage

### List All Warehouses

```terraform
data "velodb_warehouses" "all" {}

output "warehouse_count" {
  value = data.velodb_warehouses.all.total
}

output "warehouse_names" {
  value = [for wh in data.velodb_warehouses.all.warehouses : wh.name]
}
```

### Filter by Cloud Provider and Region

```terraform
data "velodb_warehouses" "beijing_saas" {
  cloud_provider  = "aliyun"
  region          = "cn-beijing"
  deployment_mode = "SaaS"
}
```

### Search by Keyword

```terraform
data "velodb_warehouses" "production" {
  keyword = "production"
}

output "prod_warehouse_ids" {
  value = [for wh in data.velodb_warehouses.production.warehouses : wh.warehouse_id]
}
```
